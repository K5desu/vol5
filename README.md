# このリポジトリについて

このリポジトリは[サポーターズ](https://talent.supporterz.jp/)主催のハッカソン vol5 の即席チーム 20 での開発用リポジトリです。

---

# 始め方

1. `npm i`で必要なライブラリをインストールします

2. 以下二つのコマンドをターミナルで実行します

```bash
     cd src/python
     uvicorn index:app --reload

```

3. アプリケーションを実行します

```bash
   npm run dev
```

or

```bash
npm run build
npm run start
```
