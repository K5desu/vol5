# 始め方

1. `npm i`で必要なライブラリをインストールします

2. 以下二つのコマンドをターミナルで実行します

```bash
     cd src/python
     uvicorn index:app --reload

```

3. アプリケーションを実行します

```bash
   npm run dev
```

or

```bash
npm run build
npm run start
```
